## 部署应用

参考下面的教程，投入部署应用的实践活动，快速掌握 Websoft9。

- [使用模板一键部署应用](./appstore-guide)
- [使用程序环境部署应用](./runtime)

## 系统管理

如果您是系统管理员，需要确保系统的下限，那么如下几个系统管理相关的任务是有必要的：

- [升级](./upgrade)：如果安装的版本不是最新版，建议升级后再使用
- [备份](./backup)：从一个开始就设置备份，是使用软件的良好习惯。
