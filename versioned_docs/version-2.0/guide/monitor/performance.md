# 应用性能实时监控

Websoft9 可以针对应用程序的**所有容器**进行实施的性能监测：

1. 登录到 Websoft9 控制台后，通过 **我的应用** 菜单找到所需监控的应用

2. 进入 **容器** 标签页，在操作栏中点击 **ⵏꓲ￨**
   ![](./assets/websoft9-container-perf-websoft9.png)

3. 系统显示容器的资源占用情况
   ![](./assets/websoft9-container-usage-websoft9.png)
