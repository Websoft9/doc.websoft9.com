# 集成第三方网关

虽然 Websoft9 网关模块为应用访问和控制，提供了便捷的操作。但仍然有特殊业务场景，需要集成第三方网关和负载均衡配套使用。   

Websoft9 平台在设计之初，将网关打造成可替换、可集成，充分尊重用户的需求。
