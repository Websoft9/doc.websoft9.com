## 入门指南{#guide}

### 注册到  CrowdSec 官方控制台

1. [CrowdSec官网](https://app.crowdsec.net/security-engines)注册账号

2. 登陆官网后，**Security Engines > Installation >** 下复制密钥注册命令：`sudo cscli console enroll -e context <your_key>`

3. 进入 CrowdSec 容器执行2的命令后，即可查看全局攻击统计
