## 入门指南{#guide}

### 初始化{#wizard}

1. Websoft9 控制台安装 Rancher 后，在**网关**设置域名和配置 SSL 证书（必须通过 HTTPS 访问）

2. 通过 **我的应用** 查看应用详情，在 **访问** 标签页中获取访问 URL 和 **获取管理员密码** 的命令

3. **Websoft9 控制台 > 终端** 执行获取密码的命令

4. 本地浏览器输入访问 `https://URL`，输入密码后根据向导完成初始化即可使用
