## 入门指南{#guide}

### 访问 Bull Queue{#console}

1. Websoft9 控制台安装 Firecrawl 后，通过 **我的应用** 查看应用详情，在 **访问** 标签页中获取访问 URL

2. 浏览器访问 `URL/admin/<BULL_AUTH_KEY>/queues`，进入 UI 界面

    > <BULL_AUTH_KEY> 是.env中环境变量 BULL_AUTH_KEY 的值，默认是 CHANGEME
