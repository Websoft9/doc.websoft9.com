## 入门指南{#guide}

### 初始化{#wizard}

1. 登录到 Websoft9 控制台，点击左侧的 **网关** 菜单[为 Azimutt 应用设置 HTTPS 访问](./domain-https#console)（必要操作，否则无法使用）

2. 通过 **我的应用** 查看应用详情，在 **访问** 标签页中获取访问 URL

3. 浏览器访问 URL，完成初始化即可使用
