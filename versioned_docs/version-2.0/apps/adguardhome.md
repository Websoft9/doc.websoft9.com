## 入门指南{#guide}

### 初始化{#wizard}

1. Websoft9 控制台安装 AdguardHome 后，通过 **我的应用** 查看应用详情，在 **访问** 标签页中获取登录信息。  

2. 浏览器访问下面url进入面板页面，根据提示完成初始化
   - http://your_domain_or_ip:DASHBOARD_PORT

3. 初始化完成后，面板页面不能再访问，浏览器输入访问标签页中的 URL 可进入后台页面
