## 入门指南{#guide}

### 初始化{#wizard}

1. Websoft9 控制台安装 BunkerWeb 后，通过 **我的应用** 查看应用详情，在 **访问** 标签页中获取访问 URL  

2. 为 BunkerWeb 绑定域名，并设置 HTTPS 访问（**必须**）

3. 本地浏览器访问域名，完成初始化即可使用
