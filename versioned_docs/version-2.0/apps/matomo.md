## 入门指南{#guide}

### 初始化{#wizard}

Websoft9 控制台安装 Matomo 后，通过 "我的应用" 查看应用详情，在 "访问" 标签页中获取登录信息。  

1. 使用本地电脑浏览器访问 URL，进入初始化安装向导

2. 根据向导依次完成后续步骤：设置数据库连接（建议采用默认值）、创建管理账号、增加首个被监控的网站
  
3. 登录后台，获取被监控网站的 JS 脚本
  ![](./assets//matomo-backend-websoft9.png)
