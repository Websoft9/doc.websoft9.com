## 入门指南{#guide}

### Agent 授权

1. Websoft9 控制台安装 GoCD 后，通过 **我的应用** 查看应用详情，在 **访问** 标签页中获取访问 URL

2. 本地浏览器访问 URL，点击 **AGTENS**

3. 选定 AGENT 后，点击 **ENABLE**后，状态变为 Idle 即可用，可以开始运行流水线了
