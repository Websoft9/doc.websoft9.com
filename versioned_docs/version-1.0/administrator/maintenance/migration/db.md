# 数据库迁移

组织可能出于以下原因进行数据迁移：

* 建立新的数据仓库
* 检修整个系统
* 升级数据库
* 从收购中合并新数据
* 部署附加系统  

数据库迁移很常见的运维工作，因此围绕数据库迁移有大量的工具和手段，具体参考数据库文档中相关章节。

* [MySQL/MariaDB 迁移](../mysql/admin#migration)
* [PostgreSQL 迁移](../postgresql/admin)
