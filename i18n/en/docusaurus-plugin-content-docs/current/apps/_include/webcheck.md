[Web-Check](https://web-check.xyz) is **Web site security and performance testing tools**, used for Full Monitoring . Web-Check is a website inspection tool that provides security, performance and usability analysis to help users optimize their website experience.


![gui](https://libs.websoft9.com/Websoft9/DocsPicture/zh/webcheck/webcheck-gui-websoft9.png)


## Prepare

When referring to this document to use **Web-Check**, please read and ensure the following points:

- [Login to Websoft9 Console](./login-console) and find or install Web-Check:
  - Go to **My Apps** listing applications 
  - Go to **App Store** installing target application

- This application is installed by Websoft9 console.


- The purpose of this application complies with the [MIT](https://opensource.org/licenses/MIT) open source license agreement.


- [Configure the domain name](./domain-set) or **server security group** opens external network ports for application access.
