# 故障处理


我们对故障的定义：除了软件报错之外，一切给用户使用带来困扰的情景都可称之为故障（或问题）。  


* 如果您是[付费客户](./docs/business/)，请随时联系我们的 [技术支持](./helpdesk) 创建 **专属服务群**，我们会组织专家团队帮助您解决故障。  

* 如果您是开源用户，请到我们的 Github 上的[开源项目](https://github.com/websoft9)中提交 Issue 获取支持。
