[MonkeyCode](https://monkeycode-ai.com) is **AI Programming Assistant**, used for AI Agents/Chatbots/RAG AI DevOps . MonkeyCode is an enterprise-grade AI programming assistant designed specifically for R&D collaboration and R&D management scenarios.


![gui](https://libs.websoft9.com/Websoft9/DocsPicture/zh/monkeycode/monkeycode-gui-websoft9.png)


## Prepare

When referring to this document to use **MonkeyCode**, please read and ensure the following points:

- [Login to Websoft9 Console](./login-console) and find or install MonkeyCode:
  - Go to **My Apps** listing applications 
  - Go to **App Store** installing target application

- This application is installed by Websoft9 console.


- The purpose of this application complies with the [GPL-3.0](https://opensource.org/licenses/GPL-3.0) open source license agreement.


- [Configure the domain name](./domain-set) or **server security group** opens external network ports for application access.
