[SigNoz](https://signoz.io/) is **open-source observability platform**, used for APM Data Insights & BI . SigNoz is an open-source observability platform native to OpenTelemetry with logs, traces and metrics in a single application.


![gui](https://libs.websoft9.com/Websoft9/DocsPicture/zh/signoz/signoz-gui-websoft9.png)


## Prepare

When referring to this document to use **SigNoz**, please read and ensure the following points:

- [Login to Websoft9 Console](./login-console) and find or install SigNoz:
  - Go to **My Apps** listing applications 
  - Go to **App Store** installing target application

- This application is installed by Websoft9 console.


- The purpose of this application complies with the [MIT](https://opensource.org/licenses/MIT) open source license agreement.


- [Configure the domain name](./domain-set) or **server security group** opens external network ports for application access.
