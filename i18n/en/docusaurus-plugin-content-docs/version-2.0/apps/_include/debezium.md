[Debezium](https://debezium.io) is **an open source distributed platform for change data capture**, used for Data Orchestration and Integration Real-time Streaming Analysis . Debezium is an open-source tool that captures database change streams, enabling low-latency data change event streams to facilitate real-time data synchronization.

## Prepare

When referring to this document to use **Debezium**, please read and ensure the following points:

- [Login to Websoft9 Console](./login-console) and find or install Debezium:
  - Go to **My Apps** listing applications 
  - Go to **App Store** installing target application

- This application is installed by Websoft9 console.


- The purpose of this application complies with the [apache2](https://opensource.org/licenses/Apache-2.0) open source license agreement.


- [Configure the domain name](./domain-set) or **server security group** opens external network ports for application access.
