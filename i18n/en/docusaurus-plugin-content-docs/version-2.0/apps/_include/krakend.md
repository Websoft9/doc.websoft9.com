[KrakenD](https://www.krakend.io) is **Open source API gateway**, used for API Gateway . Krakend is a high-performance API gateway that supports a microservices architecture and provides routing, aggregation, authentication and rate limiting.


![gui](https://libs.websoft9.com/Websoft9/DocsPicture/zh/krakend/krakend-gui-websoft9.png)


## Prepare

When referring to this document to use **KrakenD**, please read and ensure the following points:

- [Login to Websoft9 Console](./login-console) and find or install KrakenD:
  - Go to **My Apps** listing applications 
  - Go to **App Store** installing target application

- This application is installed by Websoft9 console.


- The purpose of this application complies with the [apache2](https://opensource.org/licenses/Apache-2.0) open source license agreement.


- [Configure the domain name](./domain-set) or **server security group** opens external network ports for application access.
