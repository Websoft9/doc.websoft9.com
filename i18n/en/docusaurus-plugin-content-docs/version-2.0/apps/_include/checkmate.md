[Checkmate](https://checkmate.so/) is **Website monitoring tool**, used for Full Monitoring . Checkmate is an open-source, self-hosted tool designed to track and monitor server hardware, uptime, response times, and incidents in real-time with beautiful visualizations.


![gui](https://libs.websoft9.com/Websoft9/DocsPicture/zh/checkmate/checkmate-gui-websoft9.png)


## Prepare

When referring to this document to use **Checkmate**, please read and ensure the following points:

- [Login to Websoft9 Console](./login-console) and find or install Checkmate:
  - Go to **My Apps** listing applications 
  - Go to **App Store** installing target application

- This application is installed by Websoft9 console.


- The purpose of this application complies with the [GPL-3.0](https://opensource.org/licenses/GPL-3.0) open source license agreement.


- [Configure the domain name](./domain-set) or **server security group** opens external network ports for application access.
