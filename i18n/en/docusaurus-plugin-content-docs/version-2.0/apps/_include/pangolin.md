[Pangolin](https://digpangolin.com/) is **Safe tunnel tools**, used for Virtual networking . Pangolin at its core is a self-hosted tunneled reverse proxy with identity and access management, designed to securely expose private resources through encrypted WireGuard tunnels running in user space. Think self hosted Cloudflare tunnels.


![gui](https://libs.websoft9.com/Websoft9/DocsPicture/zh/pangolin/pangolin-gui-websoft9.png)


## Prepare

When referring to this document to use **Pangolin**, please read and ensure the following points:

- [Login to Websoft9 Console](./login-console) and find or install Pangolin:
  - Go to **My Apps** listing applications 
  - Go to **App Store** installing target application

- This application is installed by Websoft9 console.


- The purpose of this application complies with the [GPL-3.0](https://opensource.org/licenses/GPL-3.0) open source license agreement.


- [Configure the domain name](./domain-set) or **server security group** opens external network ports for application access.
