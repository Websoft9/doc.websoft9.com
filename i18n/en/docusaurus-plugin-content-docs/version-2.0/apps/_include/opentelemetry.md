[OpenTelemetry](https://opentelemetry.io/) is **open source telemetry framework**, used for Log Analytics APM . OpenTelemetry is a collection of APIs, SDKs, and tools. Use it to instrument, generate, collect, and export telemetry data (metrics, logs and traces) to help you analyze your software's performance and behavior.

## Prepare

When referring to this document to use **OpenTelemetry**, please read and ensure the following points:

- [Login to Websoft9 Console](./login-console) and find or install OpenTelemetry:
  - Go to **My Apps** listing applications 
  - Go to **App Store** installing target application

- This application is installed by Websoft9 console.


- The purpose of this application complies with the [apache2](https://opensource.org/licenses/Apache-2.0) open source license agreement.


- [Configure the domain name](./domain-set) or **server security group** opens external network ports for application access.
