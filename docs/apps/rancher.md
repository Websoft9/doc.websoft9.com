## 入门指南{#guide}

### 初始化{#wizard}

1. Websoft9 控制台安装 Rancher 后，在**网关**设置域名和配置 SSL 证书

2. 通过 **我的应用** 查看应用详情，在 **访问** 标签页中获取访问 URL

3. 终端执行**获取管理员密码**命令获取密码

4. 本地浏览器输入访问 URL，输入密码后根据向导完成初始化即可使用
