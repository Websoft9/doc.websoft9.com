## 入门指南{#guide}

### 创建 Repository{#wizard}

1. Websoft9 控制台安装 Gitness 后，通过 **我的应用** 查看应用详情，在 **访问** 标签页中获取登录信息。  

2. 点击 **New Project** 先创建一个 Project

3. 然后在 Project 下创建 Repository
