## 准备

在参阅本文档使用 ChatGPT Next Web 时，需要确保如下几点：

- 应用是基于 Websoft9 安装的

- 请检查应用的使用许可协议，确保符合要求

- 应用具备访问条件：[配置域名](./domain-set) 或 **服务器安全组**开启网外端口
