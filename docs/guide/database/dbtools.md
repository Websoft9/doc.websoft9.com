# 部署数据库管理工具

Websoft9 平台提供了丰富的**在线数据库工具**，支持导入导出、备份、SQL查询、可视化操作、用户管理、访问控制以及数据库审计等常用功能，确保用户能选用最适合的管理解决方案。  


## 开始部署

1. 登录 Websoft9 控制台，打开应用商店功能

2. 进入 **"数据库 > 管理与迁移** 子类目
   ![](./assets/websoft9-appstore-dbmanagelist.png)

3. 选择目标数据库工具，开始 [部署应用](./deployment#appstore)
